Django Rest Framework (DRF)

HTTP (Hyper Text Transfer Protocol) - Used to transfer information from frontend and backend

HTTP Method - CRUD Operations

POST - Creation

PUT - Updation

GET - Reading

PATCH - Partial Updation

DELETE - Deletion


Request & Response - Headers & Body

Headers - Configuration

Body - Actual information of user. (Optional - POST / PUT / PATCH)

Serializer: Converting Model instances into JSON format.

De Serializer: Converting JSON format into Model instances to store in database.

Validation: Validating incoming data.


Syntax for creating serializer:

from rest_framework import serializers

class ProductSerializer(serializers.ModelSerializer):
class Meta:
model = Product
fields = ' all '

urls.py -> views.py -> serializers.py -> models.py -> database
